macro_rules! static_guid {
    ($name:ident, $guid:tt) => { ... };
}

Defines a named static GUID.

Note that you need to prefix the numbers with 0x to indicate they're hexadecimal. Limitations in macro_rules! prevent this being done automatically.

Examples

static_guid!(STATIC_GUID, {0xAAAAAAAA-0xBBBB-0xCCCC-0xDDDD-0xEEEEEEEEEEEE});
 
// This would be equal if you could compare `GUID`s...
