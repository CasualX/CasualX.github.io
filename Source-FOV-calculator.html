Source FOV Calculator

What's the Deal?

Source follows the usual Quake-derived behavior: widescreen support scales horizontal FOV to preserve vertical FOV. That means the same config value produces a wider real view on wider displays.

This is deterministic, not subjective. If you want to inspect the implementation directly, the formula is in the inline javascript above.
